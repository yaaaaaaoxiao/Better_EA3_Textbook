# Better_EA3_Textbook
To enhance the EA3 textbook (othello.mech.northwestern.edu) of Northwestern University


# How To Use
Use the CSS file! Simply inject it for othello.mech.northwestern.edu!

The way I use it is through the Stylebot plugin. After installing the plugin in your browser, add a customize style with webpage othello.mech.northwestern.edu, then copy and paste the content of the CSS file!
*Stylebot is only for Chrome, Firefox, and Edge.
